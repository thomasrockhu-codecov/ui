# Badge

This component will render a badge component with nothing inside.

## Props

Accepts `HTMLDivElement` props, with two caveats:

| name            | description | default               | comments                                                          |
| :-------------- | :---------- | :-------------------- | :---------------------------------------------------------------- |
| backgroundColor | colorFn     | theme.color.cta       |                                                                   |
| color           | colorFn     | theme.color.textLight | default color is set if children is a not a component or function |

## Usage

Basic usage

<Preview>
  <Badge>1</Badge>
</Preview>
